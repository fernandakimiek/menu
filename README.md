# Menu

- HTML5 + Javascript + SASS BEM (Block-Element-Modifier)
- Mobile First
- Display Flexbox
- Display Grid
- Position Relative, Fixed and Absolute
- Opacity
- Transition

# Compile SASS

Installing dependencies node-sass `npm install`  
Compile files of SASS folder for CSS folder `npm run sass`
